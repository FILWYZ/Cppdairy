## 题 1（基础）

❓ 以下声明分别是什么意思？

```cpp
const int* p1;
int* const p2;
const int* const p3;
```

### ✅ 答案

* p1：指向 const 的指针
* p2：const 指针
* p3：指向 const 的 const 指针

---

## 题 2（接口设计）

❓ 哪个接口更好？为什么？

```cpp
void foo(const string* s);
void foo(const string& s);
```

### ✅ 面试级答案

选择：`const string&`

原因：

* 不允许空值
* 调用简单
* 语义明确

---

## 题 3（进阶）

❓ 什么时候必须用指针而不能用引用？

### ✅ 答案要点

* 参数可选（可能不存在）
* 需要表达空状态

## 🧠 Day 4 核心认知

> **优秀的 C++ 接口，是“不容易被用错的接口”。**

当你开始：

* 默认使用 const
* 优先使用引用
* 把“能否为空”当作接口问题
