## 一、概念理解题

### 题 1：RAII 的核心思想是什么？

#### 答案

* 构造函数获取资源
* 析构函数释放资源

---

### 题 2：智能指针解决了什么根本问题？

#### 答案

* 明确资源所有权
* 自动释放

---

## 二、判断题

### 题 3：所有动态对象都应该使用 shared_ptr？

#### 答案

❌ 错

应优先使用 `unique_ptr`

---

## 三、代码理解题

### 题 4：下面代码有什么问题？

```cpp
void f() {
    int* p = new int(10);
    if (cond) return;
    delete p;
}
```

#### 答案

* 早退导致内存泄漏

---

## 四、重构设计题

### 题 5：如何用 RAII 重构上题代码？

#### 答案

```cpp
void f() {
    auto p = std::make_unique<int>(10);
    if (cond) return;
}
```

---

## 五、工程思维题

### 题 6：为什么说 RAII 是异常安全的基础？

#### 答案

* 析构函数在栈展开时自动调用

---

## 六、面试总结题

### 题 7：一句话说明你如何看待旧代码重构

#### 答案示例

> 重构旧代码的本质，是用 RAII 和清晰的所有权模型，把不确定的人工约定变成编译期可验证的结构。
