本文件通过**正确示例 + 错误示例对比**，帮助你把 C++11 基础用法“用对”。

---

## 示例 1：auto 的典型正确用法

```cpp
vector<int> v = {1,2,3};
for (auto it = v.begin(); it != v.end(); ++it) {
    // it 的类型自动推导
}
```

---

## 示例 2：auto 避免类型错误

```cpp
vector<int> v;
// vector<int>::iterator it = v.begin();
auto it = v.begin();
```

---

## 示例 3：auto 引用错误示例

```cpp
for (auto x : v) {
    x++; // 修改的是副本
}
```

### 正确写法

```cpp
for (auto& x : v) {
    x++;
}
```

---

## 示例 4：nullptr 解决重载歧义

```cpp
void foo(int);
void foo(char*);

foo(nullptr); // 调用 foo(char*)
```

---

## 示例 5：range-for 遍历 map

```cpp
map<int, int> m;
for (const auto& kv : m) {
    // kv.first / kv.second
}
```

---

## 示例 6：错误使用 NULL

```cpp
int* p = NULL; // ❌ 不推荐
int* q = nullptr; // ✅
```

---

## 工程实践建议

* auto + const & 是最安全组合
* range-for 默认使用 const auto&
* nullptr 是唯一合法空指针
