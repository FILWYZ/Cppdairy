通过实例理解二者在真实工程中的用法。

---

## 示例 1：constexpr 作为数组大小

```cpp
constexpr int N = 8;
int arr[N];
```

---

## 示例 2：constexpr 函数

```cpp
constexpr int add(int a, int b) {
    return a + b;
}

constexpr int x = add(2, 3);
```

---

## 示例 3：const 不能用于模板参数

```cpp
const int n = 10;
// array<int, n> a; // ❌

constexpr int m = 10;
array<int, m> b; // ✅
```

---

## 示例 4：enum class 防止隐式转换

```cpp
enum class Mode { Read, Write };

Mode m = Mode::Read;
// int x = m; // ❌
```

---

## 示例 5：enum class 作为状态机

```cpp
enum class State {
    Init,
    Running,
    Stop
};
```

---

## 示例 6：与 switch 配合

```cpp
switch (state) {
case State::Init:
    break;
case State::Running:
    break;
default:
    break;
}
```

---

## 工程实践总结

* constexpr 提供零成本抽象
* enum class 是强类型状态表达
