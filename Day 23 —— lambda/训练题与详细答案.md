## 一、基础理解题

### 题 1：lambda 的本质是什么？

#### 答案

* 一个匿名类对象
* 重载了 `operator()`

---

### 题 2：lambda 有名字吗？

#### 答案

❌ 没有

* 类型不可写出

---

## 二、捕获规则题

### 题 3：以下 lambda 捕获了什么？

```cpp
int a = 1, b = 2;
auto f = [a, &b]() { };
```

#### 答案

* `a`：值捕获
* `b`：引用捕获

---

### 题 4：为什么引用捕获有风险？

#### 答案

* 可能访问已销毁对象
* 容易产生悬垂引用

---

## 三、mutable 题

### 题 5：mutable 的作用是什么？

#### 答案

* 允许修改值捕获的副本
* 不影响外部变量

---

## 四、工程判断题

### 题 6：以下 lambda 是否安全？

```cpp
auto f;
{
    int x = 10;
    f = [&]() { return x; };
}
```

#### 答案

❌ 不安全（悬垂引用）

---

## 五、应用场景题

### 题 7：lambda 最典型的三个应用场景？

#### 答案

1. STL 算法
2. 回调 / 延迟执行
3. 局部策略逻辑

---

## 六、面试总结题

### 题 8：一句话评价 lambda 的捕获规则

#### 答案

> 捕获方式决定 lambda 的语义与安全性，默认显式捕获、优先值捕获。
