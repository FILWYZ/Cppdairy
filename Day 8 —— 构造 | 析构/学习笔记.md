### 一、什么是构造函数（Constructor）

构造函数是在 **对象创建时自动调用** 的特殊成员函数，用来完成对象的初始化。

**特点：**

* 名字与类名相同
* 没有返回值（连 void 都没有）
* 可以重载（多个不同参数版本）

```cpp
class Person {
public:
    Person() {
        cout << "默认构造函数调用" << endl;
    }
};
```

---

### 二、默认构造函数

**默认构造函数**：

* 无参数的构造函数
* 或所有参数都有默认值

```cpp
class A {
public:
    A() {}
};
```

⚠️ 注意：

* 一旦你自己写了构造函数，**编译器就不会再自动生成默认构造**

---

### 三、拷贝构造函数（Copy Constructor）

拷贝构造函数用于 **用一个已有对象初始化另一个对象**。

**函数形式：**

```cpp
类名(const 类名& 对象)
```

```cpp
class A {
public:
    int x;

    A(int x) : x(x) {}

    A(const A& other) {
        x = other.x;
        cout << "拷贝构造函数调用" << endl;
    }
};
```

**触发场景：**

1. 用一个对象初始化另一个对象
2. 函数按值传参
3. 函数按值返回对象

---

### 四、析构函数（Destructor）

析构函数在 **对象生命周期结束时自动调用**，用于资源释放。

**特点：**

* 名字是 `~类名`
* 无参数、无返回值
* 一个类只能有一个析构函数

```cpp
class A {
public:
    ~A() {
        cout << "析构函数调用" << endl;
    }
};
```

---

### 五、对象生命周期顺序

**基本原则：**

> 构造顺序 = 创建顺序
> 析构顺序 = 创建顺序的反向

```cpp
int main() {
    A a;
    A b;
}
```

执行顺序：

```
构造 a
构造 b
析构 b
析构 a
```
