## 一、constexpr 理解题

### 题 1：constexpr 一定在编译期执行吗？

#### 答案

❌ 不一定

* 满足条件时可在编译期求值

---

### 题 2：constexpr 与 const 的核心区别？

#### 答案

* constexpr 保证编译期常量
* const 只保证不可修改

---

## 二、constexpr 函数题

### 题 3：以下函数是否是 constexpr？

```cpp
constexpr int f(int x) {
    return x + 1;
}
```

#### 答案

✅ 是

---

## 三、enum class 题

### 题 4：为什么推荐 enum class？

#### 答案

* 强类型
* 无隐式 int 转换
* 作用域安全

---

### 题 5：enum class 默认能转 int 吗？

#### 答案

❌ 不能（需显式 cast）

---

## 四、工程判断题

### 题 6：以下写法是否推荐？

```cpp
enum Color { Red, Green };
```

#### 答案

❌ 不推荐（应使用 enum class）

---

## 五、综合题

### 题 7：constexpr + enum class 的典型使用场景？

#### 答案

* 编译期配置值
* 强类型状态 / 策略标识

---

## 六、面试总结题

### 题 8：一句话总结 constexpr / enum class

#### 答案

> constexpr 把计算前移到编译期，enum class 把枚举变成强类型对象，二者是现代 C++ 安全与性能的基石。
