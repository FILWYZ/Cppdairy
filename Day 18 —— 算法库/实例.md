通过示例理解 `sort / find / transform` 的正确使用方式。

---

## 示例 1：`sort` 基本排序

```cpp
#include <algorithm>
#include <vector>
#include <iostream>
using namespace std;

int main() {
    vector<int> v = {3,1,4,2};
    sort(v.begin(), v.end());
    for (int x : v) cout << x << ' ';
}
```

---

## 示例 2：自定义比较器

```cpp
sort(v.begin(), v.end(), greater<int>());
```

或：

```cpp
sort(v.begin(), v.end(), [](int a, int b){
    return a > b;
});
```

---

## 示例 3：错误示例 —— 对 list 使用 sort

```cpp
list<int> lst = {3,1,2};
// sort(lst.begin(), lst.end()); // 编译失败
lst.sort(); // 正确
```

---

## 示例 4：`find` 查找元素

```cpp
#include <algorithm>
#include <vector>
#include <iostream>
using namespace std;

int main() {
    vector<int> v = {1,2,3,4};
    auto it = find(v.begin(), v.end(), 3);
    if (it != v.end()) cout << "found";
}
```

---

## 示例 5：错误示例 —— 用算法 find 查 map

```cpp
map<int,int> m;
// find(m.begin(), m.end(), 3); // 低效
m.find(3); // O(log n)
```

---

## 示例 6：`transform` 单输入区间

```cpp
#include <algorithm>
#include <vector>
#include <iostream>
using namespace std;

int main() {
    vector<int> v = {1,2,3};
    vector<int> out(v.size());

    transform(v.begin(), v.end(), out.begin(), [](int x){
        return x * x;
    });

    for (int x : out) cout << x << ' ';
}
```

---

## 示例 7：`transform` 就地变换

```cpp
transform(v.begin(), v.end(), v.begin(), [](int x){
    return x + 1;
});
```

---

## 示例 8：双输入区间 transform

```cpp
vector<int> a = {1,2,3};
vector<int> b = {4,5,6};
vector<int> c(3);

transform(a.begin(), a.end(), b.begin(), c.begin(), plus<int>());
```

---

## 工程实践建议

* 排序前确认容器类型
* 查找优先使用容器成员函数
* `transform` 前确保输出区间大小
