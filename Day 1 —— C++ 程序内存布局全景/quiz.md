## 题目 1

```cpp
int* foo() {
    int x = 10;
    return &x;
}
```

❓ 问题：这段代码有什么问题？

### ✅ 答案解析

* `x` 是栈变量
* 函数返回后栈帧销毁
* 返回地址变为悬空指针（UB）

## 题目 2

```cpp
void foo() {
    static int x = 0;
    x++;
}
```

❓ 多次调用 `foo()`，`x` 的值如何变化？

### ✅ 答案解析

* `x` 在静态区
* 只初始化一次
* 多次调用会累加

## 题目 3（后端面试题）

❓ 为什么后端服务中应尽量减少全局变量？

### ✅ 答案要点

* 生命周期过长
* 线程不安全
* 不利于测试和维护
