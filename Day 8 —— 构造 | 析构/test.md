### 题目 1

下面代码会输出什么？

```cpp
class A {
public:
    A() { cout << "A"; }
    ~A() { cout << "~A"; }
};

int main() {
    A a;
}
```

**答案：**

```
A~A
```

**解析：**

* 创建对象调用构造
* 离开作用域调用析构

### 题目 2

下面哪种情况会调用拷贝构造？

A. `A a;`
B. `A b = a;`
C. `void f(A a)`

**答案：**

* ✅ B
* ✅ C

### 题目 3（思考题）

析构函数为什么不能有参数？

**解析：**

* 析构函数由系统自动调用
* 系统无法传递参数
* 会破坏对象生命周期管理
